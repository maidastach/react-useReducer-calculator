## Run Locally

Clone the project
```bash
  git clone https://github.com/maidastach/useReducer-calculator
```

Go to the project directory
```bash
  cd useReducer-calculator
```

Install dependencies
```bash
  npm install
```

Start the React server
```bash
  npm run start
```

Run the application at
```bash
  http://localhost:3000
```
